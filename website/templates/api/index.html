This API provides JSON Data

There are two URLs to get data from the API:


/api/rooms

It provides a list of all the rooms in the DataBase

The data is in this format:

{ 'len': xx, 'data': { 'room1': { 'surface': xx, 'volume': yy }, 'room2': { 'surface': xx, 'volume': yy }, ..., 'roomN': { 'surface': xx, 'volume': yy }, } }


/api/data/room=XXXX

It provides data for a specific room

The data is in this format:

{ 'len': xx, 'data': { 'type1': val1, 'type2': val2, ... , 'typeN': valN } } }
