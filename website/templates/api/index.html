This API provides JSON Data

There is currently one URL to get data from the API:


/api/rooms

It provides a list of all the rooms in the DataBase

The data is in this format:

{
   'len': xx,
   'data': [
      {
         'room_name': room1,
         'surface': surf1,
         'volume': vol1,
         'var11': val11,
         'var12': val12,
          ...,
         'var1M': val1M
      }, {
         'room_name': room2,
         'surface': surf2,
         'volume': vol2,
         'var21': val21,
         'var22': val22,
          ...,
         'var2M': val2M
      },
       ...,
      {
         'room_name': roomN,
         'surface': surfN,
         'volume': volN,
         'varN1': valN1,
         'varN2': valN2,
          ...,
         'varNM': valNM
      }
   ]
}
